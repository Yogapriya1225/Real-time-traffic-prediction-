About TrafficFlow

TrafficFlow is a comprehensive web application designed to help predict, analyze, and visualize traffic patterns. Using a combination of numeric data (vehicle counts and average speeds) and Natural Language Processing (NLP), our system provides accurate traffic level estimations ranging from Low to High.

Features

  • Multi-criteria prediction using vehicle density and speed.
  • NLP analysis to extract traffic conditions from plain text inputs.
  • Real-time traffic mapping using Leaflet and OpenStreetMap.
  • Visual analytics with interactive charts powered by Chart.js.
  • Secure user authentication and personalized history tracking.

Tech Stack

Built with Python and Flask on the backend, using SQLite for lightweight data storage. The frontend is crafted using vanilla HTML, CSS, and JS, featuring a modern, responsive, light theme design for maximum performance and user experience.
